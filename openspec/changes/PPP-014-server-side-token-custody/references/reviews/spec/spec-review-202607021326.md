## Minor Issues

### Issue 1: 新規追加の「保管トークンの削除（切断）」要件に対応する検証タスクがない
- **Severity**: Minor
- **Location**: `tasks.md`（section 9 動作検証）
- **Line**: 52-57
- **Description**: 前回指摘の解消として `specs/credential-custody/spec.md` L65-80 に「保管トークンの削除（切断）」要件（Scenario 2 件: 切断で当該 SNS トークンが削除される / 切断は他の SNS のトークンに影響しない）が追加され、実装タスク 2.4・2.5 も用意された。しかし section 9 の動作検証タスク（9.1〜9.6）には切断時の削除挙動を検証する項目がない。とくに「同一セッションの他 SNS トークンを削除しない」という安全側の不変条件は、実装ミスがセキュリティ・可用性に直結するため検証タスクで担保するのが望ましい。
- **Suggestion**: section 9 に「9.7 ある SNS を切断すると当該セッション×当該 SNS の保管トークンのみが D1 から削除され、他の SNS のトークンと接続状態が保持されることを確認」を追加する。
