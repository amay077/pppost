| 7 | proposal/tasks のフィールド名と実コードの不一致 | Minor | **修正済み** - tasks 1.1 / 2.3 で body フィールド `images`（`user_id` / `token` / `text` と同階層）と Threads API パラメータ（`image_url` / `media_type` / `access_token`）を明示的に区別して記述 |
