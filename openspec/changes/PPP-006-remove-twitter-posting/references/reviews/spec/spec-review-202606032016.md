## Summary

フロントエンドからの Twitter (X) 投稿機能廃除を扱う proposal。openspec validate --strict はパスし、Why/What Changes/Impact、Requirement の SHALL/SHALL NOT、各 Scenario が揃っている。proposal・tasks に列挙された削除対象は実コードのリファレンスと正確に一致しており、温存方針（バックエンド・twitter-text・localStorage）も明文化されている。Critical/Major の指摘はなく、軽微な改善提案（Minor）のみ。

## Issues

### Issue 1: `SettingDataType` 条件型の else 分岐が SettingDataTwitter をフォールバックに使用している
- **Severity**: Minor
- **Location**: `tasks.md`（タスク 4.2）/ 実コード `frontend/src/lib/func.ts` L38-41
- **Line**: tasks.md 30
- **Description**: 現状の `SettingDataType` は `T extends 'mastodon' ? SettingDataMastodon : T extends 'bluesky' ? SettingDataBluesky : SettingDataTwitter` という構造で、最後の else 分岐（'bluesky' でも 'mastodon' でもない場合）が `SettingDataTwitter` になっている。タスク 4.2 は「SettingType・SettingDataType から twitter を除去」とあるが、単純に `SettingDataTwitter` 文字列を消すだけでは条件型が壊れる。else 分岐を `SettingDataBluesky`（または `never`）へ置き換える必要がある点が、タスク記述からは読み取りにくい。
- **Suggestion**: タスク 4.2 に「`SettingDataType` の else 分岐（フォールバック）を `SettingDataBluesky` へ置き換える」旨を明記し、`SettingType = SettingData['type']` がユニオン縮小により自動的に `'mastodon' | 'bluesky'` になることを補足するとよい。

### Issue 2: config.ts の `post_targets.twitter` がオブジェクト型である点の確認
- **Severity**: Minor
- **Location**: `tasks.md`（タスク 4.3）/ 実コード `frontend/src/config.ts` L9-12, L32-34
- **Line**: tasks.md 31
- **Description**: タスク 4.3 は妥当だが、`post_targets.twitter` は型定義 (L9-12) と `Config` 生成側の実体 (L32-34) の両方に存在する。「型と参照を削除」という記述で両方を指していると解釈できるが、コメントアウト済みの旧 config（L56-57, L79 付近）には触れていない。これらは死コードのため必須ではないが、温存方針に含まれるのか明示があると親切。
- **Suggestion**: タスク 4.3 を「型定義 (ConfigType.post_targets.twitter) と Config 実体生成の twitter ブロックの双方を削除」と具体化。コメントアウト済み旧 config は対象外で問題なし（任意で整理）。

### Issue 3: 検証タスクに lint/型チェック以外の手動確認手順の合否基準
- **Severity**: Minor
- **Location**: `tasks.md`（タスク 6.2〜6.4）
- **Line**: tasks.md 41-43
- **Description**: 6.1 の `npm run build` 成功は機械的に検証可能で良い。6.2〜6.4 は手動確認で、Scenario と1対1で対応しており網羅性は十分。指摘ではなく確認だが、6.2「Twitter が表示されない」は接続 UI（TwitterConnection）非表示も含む点を明示するとシナリオ（接続 UI も表示されない）と完全一致する。
- **Suggestion**: 6.2 を「投稿対象チェックボックスと接続 UI の双方が表示されないこと」と補足（任意）。
